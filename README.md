## 📌 프로젝트 소개

**Simcheong2go**는 시각장애인 사용자가 SNS 콘텐츠를 더 쉽게 탐색하고,  
게시글 내용을 음성으로 들을 수 있도록 만든 **음성 기반 SNS 모바일 앱**입니다.

기존 SNS는 이미지, 텍스트, 복잡한 UI 중심으로 구성되어 있어  
시각장애인 사용자에게 접근성이 떨어질 수 있습니다.

이 프로젝트는 **TTS 기반 콘텐츠 읽기**, **단순한 화면 흐름**,  
**접근성 중심 UI**를 통해 더 편리한 SNS 경험을 제공하는 것을 목표로 합니다.
